Christmas Tiles

Christmas Tiles is a simple tile matching game for Windows.

Requirements

  • Windows (2000 or Newer)
  • Web Browser that has at least basic HTML5 support (Required for Help)
    • Windows 2000 users can get this via Firefox 12

Features

  • Custom Tilesets
  • Per Monitor DPI Compatible (Coming soon)
  • Full Language Support

The Game supports the following operating systems:

  • Windows 11
  • Windows 10
  • Windows 8 / Windows 8.1
  • Windows 7
  • Windows Vista*
  • Windows XP**
  • Windows 2000**

*Untested.

**Windows 2000 and XP support is a work in progress, but appears to be working fine.

The program may run on newer versions of Windows, but those are untested.

Playing the Game

This is the main window of the game. All you have to do is click tiles and find matching ones.

Score is how many points you have earned so far.

You start off with 50 points, and lose 1 point for every second it takes you to find a match, and 5 for every incorrect match. It resets back to 50 when you find a match

Time is how long you have been playing.

You can click the Pause button to pause the game, and then click it again to unpause it.
